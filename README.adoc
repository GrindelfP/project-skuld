= Project "Skuld" =

Preamble: Skuld is a Norn (a fate goddess) from Scandinavian mythology. My choice when picking her name as a title for my project as well as (probably) my numerical integration library is dictated by the nature of the goddess: she foretells what should happen the same as a neural network foretell the function for input parameters, not calculates it. Also, the reason is that I like the Scandinavian mythology pretty much.
